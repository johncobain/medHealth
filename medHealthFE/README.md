# medHealthFE

Frontend do MedHealth: login, dashboard, gerenciamento de médicos, pacientes e consultas, configurações de conta. Consome a API via Gateway ou diretamente.

**Stack:** React, Vite, React Router, Axios, React Toastify, Lucide React.

- Rodar local: `./app.sh run frontend` (a partir da raiz do repositório).
- Ou: `npm install` e `npm run dev` dentro de `medHealthFE`.
- Ver [README principal](../README.md) para formas de execução e estrutura do projeto.
